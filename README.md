# Калькулятор личного бюджета 3.0

Простой веб-сайт для учёта личного, семейного или проектного бюджета. Работает полностью в браузере — без сервера, регистрации и установки.

## Возможности

- учёт доходов, обязательных и переменных расходов;
- накопления и финансовые цели;
- сравнение плана и факта;
- лимиты по категориям с визуальными индикаторами;
- диаграмма расходов и история по месяцам;
- автосохранение в `localStorage`;
- экспорт и импорт CSV, печать отчёта, копирование итога.

## Быстрый старт

### Вариант 1: открыть файл

1. Откройте `index.html` в браузере.
2. Нажмите «Открыть калькулятор».

### Вариант 2: локальный сервер (рекомендуется)

```bash
npm install
npm start
```

Сайт откроется по адресу `http://localhost:3000`.

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- Canvas API для диаграммы
- LocalStorage для хранения данных

## Данные и конфиденциальность

Все данные сохраняются только в браузере на вашем устройстве. При очистке данных браузера или смене устройства история может быть потеряна — используйте экспорт CSV для резервного копирования.
